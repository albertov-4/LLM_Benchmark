# Archive

This folder contains historical and support material collected during the
project. It is intentionally outside `Benchmark_Framework/` so the reusable
benchmark path stays focused on execution.

## Contents

```text
archive/
|-- Papers/              project papers and PDFs
|-- Planning Domains/    original planning-domain collections used for selection
|-- References/          external repositories and reference implementations
`-- Quick PDDL guide.md  project note for understanding PDDL basics
```

The runner does not depend on this folder. Use it for background reading,
domain provenance, or historical comparison only.
