# Transfer Files From Your Local Machine To The Leonardo Cluster Node 

## 1. Single File Transfer to Leonardo
From your laptop (outside the Leonardo environment), copy files to the cluster:
```bash
scp local_file.py <username>@login.leonardo.cineca.it:~/LLM-Needs-a-Plan/
```

## 2. Transfer An Entire Folder 
```bash
scp -r local_folder <username>@login.leonardo.cineca.it:~/leonardo_work/IscrC_ArtLLMs/LLM-Needs-a-Plan/
```
