# Farmland

Questa famiglia di task contiene problemi PDDL basati su spostamenti tra farm
e vincoli numerici sulle risorse disponibili.

Struttura:
- `domain/domain.pddl`: definizione delle azioni e dei predicati del dominio
- `easy/`: istanze iniziali a complessita ridotta
- `medium/`: istanze intermedie
- `hard/`: istanze piu difficili

Nota:
- le istanze devono restare confrontabili tra modelli
- ogni file `.pddl` rappresenta un singolo caso di benchmark
- la difficolta dovrebbe essere documentata quando vengono aggiunte nuove istanze
