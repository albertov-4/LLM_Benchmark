# IPC 2023 Planning Domains

This folder stores the IPC 2023 planning-domain collection used as source
material during benchmark task selection. It is archive material, not part of
the runnable `Benchmark_Framework` task-discovery path.

Use these files for domain provenance, comparison, or future task curation. To
make a domain executable by the benchmark, copy the selected material into
`Benchmark_Framework/tasks/<task_family>/` and follow the layout documented in
`Benchmark_Framework/tasks/README.md`.
