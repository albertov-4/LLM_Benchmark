# Work Directory And LLMs Download

## 1. Workspaces 

### Navigate to Your Home Directory

Once you acccess the cluster, you are in a empty environment and you can start explore and modifying it.

```bash
# Once you are into the cluster check current location
pwd

# Go to home directory
cd ~

# Create projects directory if it doesn't exist
mkdir -p ~/PROJECTS
cd ~/PROJECTS
```

### Explore the working directory

It is recommended to get into the working directory in order to store and run experiments.

```bash
# Access the working directory
cd $WORK

# Verify the directory structure
ls -la

# Check the current directory 
pwd
```

## 2. Models Downloading 

### Create Hugging Face Account and Token

1. Visit [https://huggingface.co/](https://huggingface.co/) and create an account
2. Go to Settings → Access Tokens
3. Create a new token with "Read" permissions (or "Write" if you plan to upload models)
4. Copy the token (format: `hf_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx`)

### Install the dependencies 

```bash
# Load Python module 
module load python 

# create a virtual environment for the upcoming steps 
python -m venv hf_venv

# Activate the environment 
source hf_venv/bin/activate

# Install Hugging Face package 
pip install huggingface_hub

# login into Hugging Face  
hf auth login

# You'll be prompted to enter your token
# Paste your token when prompted (it won't be visible while typing)

# Now we are ready to download the models 
```

### Request Access to Gated Models

1. Visit the model page (e.g., [meta-llama](https://huggingface.co/meta-llama))
2. Click "Request Access"
3. Accept the license terms
4. Wait for approval (usually within minutes to hours)

## 3. Downloading Llama 3.1 8B Instruct (Gated Model)

### Prerequisites
- Hugging Face account with approved access to Llama models
- Valid HF token with read permissions
- Sufficient disk space (~16GB for Llama 3.1 8B)

### Using huggingface-cli

```bash
# Download Llama 3.1 8B using CLI with resume support
huggingface-cli download meta-llama/Llama-3.1-8B-Instruct --local-dir LLM-Needs-a-Plan/src/models/Llama3
```

### Verify Llama 3.1 Download

```bash
# Check downloaded files
ls -la LLM-Needs-a-Plan/src/models/Llama3

# Check model size
du -sh LLM-Needs-a-Plan/src/models/Llama3

# Test model loading
python LLM-Needs-a-Plan/src/tests/load_models.py
```
