# LLM_Benchmark

## Abstract
This repo has been created to provide a reliable and user friendly benchmark to test and evaluate the performance of various LLMs on different planning domains. The planning domains have been classified following a rigorous algorithm that provides a numerical score and it is easily scalable and easy to understand (for further information look directly at the project report). A classification of the output provided by LLMs is also implemented, it is necessary in order to understand if a given LLM is capable of providing a reliable result by solving a given PDDL problem.
The aforementioned LLMs and the tests have been executed on Leonardo HPC so the code inside the framework is used mainly to work on that platform.
The main folders contains examples of planning domains (the one we classified using the score) that are written in PDDL, a folder containing official papers related to planning problems solved wih LLMs, a folder containing similar works done by other students in the past. The last folder among the principal ones is the Benchmark Framework which contains our work with all the scripts, the planning problems chosen for the test and the various classification done.
