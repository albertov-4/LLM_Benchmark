# First Job Submission on Leonardo

## 1. Create a Test Script
```bash
cat > gpu_test.sh << 'EOF'
#!/bin/bash
#SBATCH --account=IscrC_ArtLLMs
#SBATCH --partition=boost_usr_prod
#SBATCH --time=00:05:00
#SBATCH --gres=gpu:1
#SBATCH --output=gpu_test.out

module load python/3.11.7
source ~/LLM-Needs-a-Plan/venv/bin/activate

python - <<'EOPY'
import torch
print("GPU available:", torch.cuda.is_available())
if torch.cuda.is_available():
    print("GPU name:", torch.cuda.get_device_name(0))
EOPY
EOF
```

## 2. Submit the Job
```bash
sbatch gpu_test.sh
```
Expected output:
```
Submitted batch job <job_id>
```

## 3. Monitor Job Status
```bash
squeue -u $USER
```

## 4. Check Job Output
```bash
cat gpu_test.out
```
Example result:
```
GPU available: True
GPU name: NVIDIA A100-SXM-64GB
```
